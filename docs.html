Aurora OS Documentation

Getting Started

To get started with Aurora OS, download your preferred version from the Download page and follow the installation steps below.

Installation

  1. Download the ISO for Aurora Core or Aurora Lightweight.
  2. Flash the ISO using tools like Rufus (Windows) or Balena Etcher (Linux/macOS).
  3. Boot from the USB and follow the on-screen instructions.

Basic Usage

Aurora provides a clean and customizable desktop experience. Right-click to access desktop settings, or use the Aurora Panel to tweak performance and visuals.

Troubleshooting

If you encounter any issues, check the Changelog for recent updates or visit our Linktree for support channels.
